Класи
Класи, структури, об'єднання та інтерфейси з коротким описом.
 CAsyncFileАсинхронне зчитування файлів
 CBaseEventБазова структура для всіх подій
 CDirectoryУтилітний клас для роботи з директоріями
 CEngineБазовий клас рушія всі застосунки мають наслідуватись від нього
 CEventDispatcherДиспетчер подій для централізованої обробки системних повідомлень
 CEventKeyPressedПодія натискання клавіші
 CEventKeyReleasedПодія відпускання клавіші
 CEventMouseMovedПодія руху миші
 CEventWindowCloseПодія закриття вікна
 CEventWindowResizeПодія зміни розміру вікна
 CFileУтилітний клас для роботи з файлами
 CInfoКлас для отримання інформації про платформу, компілятор і збірку
 CInputКлас для обробки введення з клавіатури
 CPathУтилітний клас для роботи з шляхами файлів та директорій
 CTimeСтатичний клас для управління часом у додатку

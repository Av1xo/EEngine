Детальний опис

Статичний клас для управління часом у додатку

Time надає функціональність для відстеження часу між кадрами (delta time), що необхідно для незалежної від частоти кадрів анімації та фізики. Всі методи та дані статичні для глобального доступу.

Нотатки
Клас має бути ініціалізований через init() перед використанням
// В Engine::init()
Time::init();
// В Engine::run() кожен кадр
Time::update();
// Використання в логіці гри
float speed = 5.0f;
position += velocity * speed * Time::delta_time();

◆ delta_time()

float Time::delta_time ( )
static

Отримує час між поточним та попереднім кадром

Delta time використовується для забезпечення плавної анімації та руху об'єктів незалежно від частоти кадрів (FPS).

Повертає
Час у секундах, що пройшов з попереднього кадру
Нотатки
Типові значення: ~0.016 сек (60 FPS), ~0.033 сек (30 FPS)
